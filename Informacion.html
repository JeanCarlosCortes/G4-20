Socios de Negocios

Explicación de cada una de las funciones del Formulario

-Cargar Socios: Lo que hace esta funcion es el de mostrar todos los datos de los socios que se han guadado en la base de datos.

-Agregar Socios: Esta función permite agregar información de socios en la base de datos al hacer esto el ID se autoincrementa.

-Cargar Socio: Dicha funcion tiene el proposito que al dar click en el boton "Editar" se envien los datos del socio que desea editar.

-Actualizar Socio: Esta función permite actualizar los datos de una fila de datos, principalmente sera utilizado para corregir algun error de ingreso de datos.

-Eliminar Socio: Esta función permite eliminar un socio de la base de datos.

Materiales

-Cargar Materiales: Esta función permite mostrar todos los registros de los socios que se han guardado en la base de datos.

-Agregar Material: Esta función permite agregar un nuevo material en la base de datos, ya que al agregar el material no es necesario ingresar el ID porque es ta auto incrementable.

-Cargar Material: Lo que hace esta función es que al dar click en el boton "Editar" se cargan los datos de ese material en el formulario, para poderlos editar.

-Actualizar Material: Esta función permite actualizar los datos de un registro dependiendo del ID, es utilizado para cambiar algunos datos.

-Eliminar Material: Con esta función se puede eliminar un material de la base de datos.

Julio Cesar Mejia Cantarero

Pedidos

-Cargar Pedidos: Carga todos los pedidos que se han insertado en la Base de Datos. Esta función envía 2 parámetros: 1 parámetro (ID) a la función Cargar Pedido y 1 a la función Eliminar Pedido.

-Agregar Pedido: Agrega un pedido, colocando la información en el formulario: Agregar pedido. Además, es posible agregar un pedido con un ID manual, el que uno desee.

-Cargar Pedido: Esta función obtiene todos los datos de un pedido recibiendo un ID que la función Cargar Pedidos le envía. Posteriormente, carga los datos obtenidos en el formulario de Agregar Pedido.

-Actualizar Pedido: Actualiza los datos de un pedido. Una vez que los datos que se quieren editar estan cargados sobre el formulario, se puede editar la información que uno desee, menos el ID.

-Eliminar Pedido: Elimina un pedido en especifico al dar click en el botón eliminar. Recibe el ID que quiere eliminar desde la función Cargar Pedidos.
